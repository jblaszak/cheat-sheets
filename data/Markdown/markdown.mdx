### Multiple Lines in List Section
Try using soft breaks and hard breaks to change up positioning of text.
```
* List Text
  Text here continues at end of line above.
  
  Text at same indentation at line below.
```

### Paragraphs
Markdown handles spaces differently. Two types of line breaks, hard break and soft break.
Hard break creates a space between sections of text. Soft break puts text on next line.

#### Hard Break
Use empty line to create hard break.
```
Some text on a line.

Some text now in a different paragraph.
```

#### Soft Break
Use two empty spaces '  ' to create soft break.
```
Some text in a line.  
Some text now on line below.
```
